Artale is an interactive platform for artists and art enthusiasts to explore, discover, and share artworks. The app allows users to browse different types of art, filter by mood or style, and view curated galleries, making it easy to find inspiration and connect with creativity.

Key Features:

Discover artwork based on mood, style, or category
Interactive and visually appealing galleries
Responsive and user-friendly interface
Showcases various art dimensions and forms

Tech Stack:

React,
Tailwind CSS,
Node,
Express,
Mongodb
